# Zabbix Dashboard

一个现代化、美观、简约的Zabbix监控仪表板，具有完整的登录系统和丰富的监控功能。

## 特性

- **现代UI设计**：采用渐变色彩和现代化UI组件
- **完整认证系统**：安全的登录/登出功能
- **响应式布局**：适配各种设备屏幕
- **实时监控**：显示主机状态、告警等信息
- **数据可视化**：图表展示监控数据

## 页面结构

1. **登录页面** (`index.html`)
   - 用户认证
   - 服务器地址配置
   - 错误提示

2. **仪表板页面** (`dashboard.html`)
   - 监控概览统计
   - 数据可视化图表
   - 最新告警列表

3. **主机页面** (`hosts.html`)
   - 主机列表展示
   - 搜索和过滤功能
   - 主机详情查看

## 技术栈

- HTML5/CSS3/ES6
- Chart.js - 数据可视化
- Font Awesome - 图标库
- Zabbix API - 数据获取

## 安装与使用

1. 克隆或下载此项目
2. 在项目根目录执行：
   ```bash
   ./start.sh
   ```
3. 打开浏览器访问 `http://localhost:8000`
4. 使用您的Zabbix凭据登录

## 功能说明

### 登录页面
- 预填充常用字段
- 表单验证
- 错误处理
- 凭据本地存储

### 仪表板页面
- 显示主机总数、问题数、可用性等统计
- 饼图展示主机状态分布
- 趋势图表显示活动情况
- 告警列表按严重性分类

### 主机页面
- 卡片式主机列表展示
- 支持按名称搜索
- 支持按状态筛选
- 点击查看详情模态框

## 安全说明

- 登录凭据仅存储在浏览器本地存储中
- API请求使用HTTPS（如果服务器支持）
- 会话过期自动跳转到登录页
